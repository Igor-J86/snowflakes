# Snowflakes
Animated snowflakes component, written in React with TypeScript.

## Usage
```jsx
import React from "react";
import { Snowflakes } from "@igor-j86/snowflakes";

const MyComponent = () => {
  return (
    <Snowflakes />
  )
}

export default MyComponent
```

## Optional props
| Prop                          | Default             |
| ----------------------------- | ------------------- |
| id:string                     | 'snowflakes'        |
| maxSize:number                | 3                   |
| maxSpeed:number               | 0.5                 |
| snowflakes:number             | 100                 |
| color:string                  | '#dddddd'           |
